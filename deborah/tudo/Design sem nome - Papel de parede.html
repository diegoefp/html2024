A barra de ferramentas contém botões para editar o elemento selecionado. Para ir de um botão a outro, use as teclas de direita e esquerda. Para ativar o botão, aperte Enter. Para voltar à tela, aperte “Ctrl+F2”.
A tela, ou editor, é onde fica seu design. Para acessar a página, aperte Enter. Para acessar os grupos ou elementos da página, aperte Enter novamente. Aperte Tab para passar para a próxima página, grupo ou elemento. Para usar a barra de ferramentas, use a tecla “Ctrl+F1”. Se um elemento estiver selecionado, você pode movê-lo usando as teclas de seta; para girar, use as teclas “,” e “.”; para redimensionar, use as teclas de seta junto com a tecla “Ctrl”. Você pode modificar os incrementos de todos esses atalhos com a tecla “Shift”.
